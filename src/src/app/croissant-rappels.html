🔔 Notifications Firebase FCM
Chaque collègue doit ouvrir l'app et cliquer sur "Activer les notifications" pour recevoir les pushs. Son token FCM est alors enregistré dans Firestore sous subscriptions/{userId}. Lors d'un remplacement, une Cloud Function Firebase envoie automatiquement le push à tous les tokens.
